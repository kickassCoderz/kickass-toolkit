-   Debounces setter
-   No need to wrap callback with `useCallback`, because hook manages callback mutation internally.
-   Full TypeScript support.

## Basic usage

<UseDebouncedStateExampleBasic />

```jsx
import { useDebouncedState } from '@kickass-coderz/hooks'

const UseDebouncedStateExampleBasic = () => {
    const [state, setState] = useDebouncedState('', 500)

    return (
        <div>
            <div>value:{state}</div>
            <input type="text" onChange={e => setState(e.target.value)} />
        </div>
    )
}
```

## Api Reference

| Property       | Type             | Default | Description                                                                                 |
| :------------- | :--------------- | :------ | :------------------------------------------------------------------------------------------ |
| `initialState` | ` T \| (() => T` | `none`  | Same as `React.useState` initialValue. It can be any value or function which returns value. |
| `delay`        | `number`         | `none`  | Delay for debounce.                                                                         |

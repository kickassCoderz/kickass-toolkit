# {meta.title}

<p>{meta.description}</p>

-   Debounced function is allways void.
-   No need to wrap callback with `useCallback`, because hook manages callback reference internally.
-   Deferred execution automatically cancelled on component unmount.
-   Returns tuple with `execute` and `clear` methods which you dont have to wrap with `useCallback` as this is managed internally.
-   Full TypeScript support.

## Basic usage

<UseDebouncedCallbackExampleBasic />

```jsx
import { useDebouncedCallback } from '@kickass-coderz/hooks'
import { ChangeEvent, useState } from 'react'

const UseDebouncedCallbackExampleBasic = () => {
    const [state, setState] = useState('')

    const [handleChange] = useDebouncedCallback(event => {
        setState(event.target.value)
    }, 1000)

    return (
        <div>
            <div>value:{state}</div>
            <input type="text" onChange={handleChange} />
        </div>
    )
}
```

## Api Reference

### Params

| Property     | Type               | Default | Description                                  |
| :----------- | :----------------- | :------ | :------------------------------------------- |
| `callbackFn` | `CallableFunction` | `none`  | A function which execution will be deffered. |
| `delay`      | `number`           | `none`  | Delay for debounce                           |

### Returns

| Property   | Type                                  | Default | Description                                                                                                                   |
| :--------- | :------------------------------------ | :------ | :---------------------------------------------------------------------------------------------------------------------------- |
| `Array`    | `[CallableFunction,CallableFunction]` | `none`  | Returns a tuple with two functions. First one is your provided and debounced callback, other one is clear method for timeout. |
| `Array[0]` | `CallableFunction`                    | `none`  | Debounced callback trigger.                                                                                                   |
| `Array[1]` | `CallableFunction`                    | `none`  | Clear method which clears timeout.                                                                                            |
